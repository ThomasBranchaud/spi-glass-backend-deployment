# SPI-Glass Backend Server README

Welcome to the README for the SPI-Glass backend. This covers how to deploy the server on both localhost and Vercel, a common web-hosting platform. 

The majority of the code for this server is contained with the server.js file, while the varaibles that control what database the server attempts to connect to are in the databaseConnection.js file. 

This server was created using JavaScript and Express and is made to be as easy to expand as possible for future development.

# Localhost Deployment
(Should mainly be done for testing)

To deploy on localhost you need to clone this repository onto your computer and open it in your IDE of choice.
Once you open the project, open the server.js file and attempt to run it. If you are unable to do so right away, attempt the npm install command and retry.
